# Intake question bank (AskUserQuestion option sets)

Use these as starting points. Drop any question already answered. Put the recommended option
first and append " (Recommended)". Keep to 4 questions per AskUserQuestion call. The user can
always type a custom answer, so options are prompts, not limits.

## Round 1 — Essentials (almost always ask)

**Q: Scope** — header "Scope"
- Full remodel (gut + reconfigure)
- Refresh (finishes/fixtures, keep layout)
- Concept/visualization only
- Addition / new construction

**Q: Budget tier** — header "Budget"
- Better (mid-range, some custom)
- Best (high-end, fully custom) (often the firm's lane)
- Good (value-driven)
- Has a hard cap (ask the number)

**Q: Style direction** — header "Style"
- Warm modern / minimal (firm default)
- Transitional
- Contemporary / bold
- Match an uploaded reference image

**Q: Deliverables wanted** — header "Deliverables", multiSelect
- Floor plan + DXF
- 3D render(s)
- Elevations
- Bill of Materials
- Client deck/PDF

## Round 2 — Room-specific & details (ask what's relevant)

**Q: Existing conditions / plan source** — header "Plan source"
- I'll give dimensions now
- Work from the plan/photo I uploaded (field-verify)
- Measure later — use a typical size for now
- It's new construction (no existing)

**Q: Material priorities** — header "Materials", multiSelect
- Large-format stone (Calacatta / quartzite / porcelain slab)
- Custom millwork (white oak / walnut / painted)
- Brushed/aged metals
- Specific brand/product I'll name
- Avoid certain materials (I'll name)

**Q: Lighting** — header "Lighting"
- Layered + warm (2700–3000K) (firm default)
- Bright/cool task-forward
- Maximize daylight / minimal fixtures
- Show me options

**Q: Must-keep elements** — header "Keep", multiSelect
- Plumbing locations
- Windows/doors as-is
- A specific feature (island, tub, fireplace…)
- Nothing — open to reconfiguring

## Room-specific add-ons (pull as needed)
- **Kitchen:** layout (galley/L/U/island), appliance package, sink count, pantry, seating.
- **Bath:** tub (freestanding/none), shower (walk-in/curb), single vs. double vanity, water closet.
- **Primary suite:** closet scope, bed wall, sitting area, window treatments.
- **Whole home:** room priority order, flow/sightlines, consistent material palette.

## Accessibility / compliance (ask if hinted)
- Aging-in-place / ADA clearances needed? (research-only; not a compliance certification)
- Known permit or HOA constraints?

## Confirmation prompt (after writing the brief)
**Q: Ready to build?** — header "Confirm"
- Build to this brief (Recommended)
- Adjust something first (I'll say what)
